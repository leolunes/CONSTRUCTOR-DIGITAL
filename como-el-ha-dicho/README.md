# COMO ÉL HA DICHO

Aplicación web cristiana tipo PWA para leer, declarar y guardar promesas bíblicas.

---

# OBJETIVO

Ayudar a las personas a:
- fortalecer su fe,
- declarar la Palabra,
- encontrar paz,
- orar con fe,
- meditar diariamente en promesas bíblicas.

---

# TECNOLOGÍAS

- HTML5
- CSS3
- JavaScript
- JSON
- PWA (Progressive Web App)
- LocalStorage
